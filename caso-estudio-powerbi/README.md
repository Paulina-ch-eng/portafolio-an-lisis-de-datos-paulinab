# 📊 Caso de estudio: Dashboard de Desempeño Comercial | Power BI
## Descripción
Dashboard interactivo para analizar ventas 2024-2025 por período, producto y región, comparando resultados reales contra metas comerciales en pesos chilenos.
## Desafío principal
Integrar dos fuentes de datos sin relación directa por fecha: ventas con 204 registros mensuales y metas por producto, construyendo un modelo que permitiera comparar ambas métricas correctamente.
## Solución propuesta
Se limpió y transformó la data con Power Query, se creó una clave combinada por ID_Producto, Año y Mes para relacionar las tablas con cardinalidad uno a muchos, y se desarrollaron medidas DAX para calcular KPIs clave.
## Herramienta técnica utilizada
Power BI Desktop, Power Query, DAX y Excel/CSV como fuentes de datos.
## Aprendizajes
Modelado de datos con relaciones uno a muchos, transformación de datos con Power Query, creación de medidas DAX para inteligencia de tiempo y diseño de dashboards interactivos con navegación entre páginas.
## Métricas de impacto
Cumplimiento de metas del 80% sobre $79,07 millones presupuestados. Región líder Metropolitana con $15,18 millones. Laptop concentra el 55,63% del total de ventas. Crecimiento desde $28,72 millones a $63,26 millones respecto al año anterior.
## Habilidades técnicas aplicadas
Análisis de datos, modelado relacional, visualización de datos, pensamiento analítico y uso de lenguaje DAX.
## Justificación
Elegí este proyecto porque representa mi capacidad para transformar datos de múltiples fuentes en información visual y accionable para la toma de decisiones comerciales.
