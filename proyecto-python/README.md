# 📊 Análisis de Hábitos Saludables en Universitarios

## 🔹 Descripción
Este proyecto analiza los hábitos de sueño, alimentación y actividad física en jóvenes universitarios mediante técnicas de estadística descriptiva e inferencial utilizando Python.

---

## 🔹 Objetivo
Identificar patrones en los hábitos de los estudiantes y evaluar si cumplen con recomendaciones de salud (horas de sueño y actividad física).

---

## 🔹 Desafío
Trabajar con múltiples variables y aplicar conceptos estadísticos como distribuciones, intervalos de confianza y test de hipótesis para obtener conclusiones relevantes.

---

## 🔹 Herramientas utilizadas
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- SciPy  

---

## 🔹 Actividades realizadas
- Simulación de dataset  
- Limpieza y análisis de datos  
- Cálculo de probabilidades  
- Visualización de datos  
- Aplicación de inferencia estadística  
- Test de hipótesis  

---

## 🔹 Resultados
- Los estudiantes duermen en promedio menos de 8 horas  
- Se identificaron niveles moderados de estrés  
- Se observaron patrones en hábitos de salud
